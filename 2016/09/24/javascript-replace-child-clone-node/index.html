JavaScriptで要素の入れ替え | replaceChild, cloneNode

2016年9月24日

[ピュア(Vanilla)なJavaScriptでの要素の入れ替えのMEMO。
例えばAとBの要素を入れ替えたい時など。

実装方法

cloneNodeで要素のクローンを作った後に、replaceChildで置換する。
cloneNodeの引数は、nodeの子孫ノードも複製する場合はtrue
nodeのみを複製する場合はfalseとなる。
