さくっとLiveReload導入メモ

2016年9月18日

さっとローカルサーバーを立ち上げて、LiveReloadさせて作業する時のメモ。

セットアップ

まずは作業場所で、セットアップ。

package.jsonにシェルを記載。

npm run

npm run liveを実行すると、ブラウザが開いてローカルサーバーが立ち上がり、
ファイルを監視して、LiveReload開始。
Sync時に右上に表示されるお知らせは無くしたかったので、–no-notifyを付けてる。

オプションを付けたりしたい時は下記から
https://www.browsersync.io/docs/command-line#start
